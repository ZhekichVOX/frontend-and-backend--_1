# Практики 13-14

## Что реализовано

- Просмотр и добавление заметок.
- Сохранение данных в `localStorage`.
- Регистрация `Service Worker` и кэширование статики.
- Офлайн-режим (страница, список и добавление заметок доступны без сети).
- Подключен `manifest.json`, добавлены meta-теги и иконки.

## Запуск

Откройте папку через любой локальный сервер, например:

```bash
npx http-server -p 3000
```
