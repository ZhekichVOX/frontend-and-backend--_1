# Practice 6 - Real-time Chat Application

Полнофункциональное приложение чата в реальном времени с использованием Socket.IO и WebSocket.

## Возможности

✅ Реал-тайм сообщения через WebSocket  
✅ Список активных пользователей в реальном времени  
✅ Индикатор печатания  
✅ Удаление собственных сообщений  
✅ История сообщений (последние 100)  
✅ Системные сообщения о присоединении/отключении  
✅ Валидация имён пользователей (без дубликатов)  
✅ Адаптивный дизайн  

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

Откройте в браузере `http://localhost:3000`

## Как использовать

1. Откройте `http://localhost:3000` в браузере
2. Введите ваше имя пользователя
3. Нажмите "Войти"
4. Начните писать сообщения в реальном времени
5. Увидите список всех активных пользователей
6. При печатании букв вы увидите индикатор "печатает"

## Socket.IO события

### Отправляемые события (от клиента к серверу)

- `join` - присоединение пользователя с именем
- `message:send` - отправка сообщения
- `message:delete` - удаление сообщения
- `typing` - начало печатания
- `stopped:typing` - прекращение печатания

### Получаемые события (от сервера к клиенту)

- `users:update` - обновление списка пользователей
- `message:received` - получение нового сообщения
- `user:joined` - уведомление о присоединении пользователя
- `user:left` - уведомление об отключении пользователя
- `user:typing` - получение уведомления о печатании
- `user:stopped-typing` - прекращение печатания
- `message:deleted` - сообщение удалено

## API endpoints

```
GET /api/messages    - Получить историю сообщений (последние 50)
GET /api/users       - Получить список активных пользователей
```

## Технологии

- **Frontend**: HTML, CSS, JavaScript, Socket.IO Client
- **Backend**: Node.js, Express, Socket.IO, Cors

## Примечания

- Максимум 20 символов в имени пользователя
- История сообщений хранится в памяти (100 последних)
- При перезагрузке страницы история сообщений сохраняется
- При перезагрузке сервера все пользователи отключатся
